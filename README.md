# Crazy Ball

Demo final jugable del juego **Crazy Ball**, preparada para GitHub Pages.

## Incluye

- Menú principal con botón JUGAR.
- Pantalla de instrucciones.
- Juego compatible con móvil, tablet y ordenador.
- Control táctil, ratón, flechas y WASD.
- Obstáculos aleatorios: asteroides, ovnis, satélites, planetas y pinchos.
- Monedas, escudos, vidas extra y meteoritos.
- Multiplicadores aleatorios x2, x3, x4, x5, x6, x8 y x10.
- Destellos, partículas, vibración visual y efectos de impacto.
- Puntuación, niveles, vidas, monedas y récord persistente.
- Pausa, reinicio, pantalla de fin de partida y nuevo récord.
- Sin librerías ni archivos externos.

## Publicar en GitHub Pages

1. Crea un repositorio llamado `crazy-ball`.
2. Sube `index.html` y `README.md` a la raíz del repositorio.
3. Ve a **Settings > Pages**.
4. En **Build and deployment**, selecciona **Deploy from a branch**.
5. Selecciona la rama `main` y la carpeta `/ (root)`.
6. Guarda los cambios.

También puedes probarlo abriendo `index.html` directamente en un navegador.
