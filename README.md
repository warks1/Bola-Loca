# Bola Loca

Demo vertical jugable para GitHub Pages.

## Subir a GitHub
1. Sube todos los archivos y carpetas de este ZIP al repositorio.
2. En GitHub: Settings > Pages.
3. Branch: main / root.
4. Abre la URL de GitHub Pages.

## Controles
Arrastra la bola con el dedo o ratón. Recoge monedas y power-ups. Evita obstáculos rojos.
