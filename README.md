# Bola Loca v0.1 Alpha

Versión jugable HTML5 para GitHub Pages.

Incluye:
- Menú funcional
- Bola controlada con dedo/ratón
- HUD con puntos, monedas, récord y multiplicador
- Obstáculos rojos
- Monedas amarillas
- Meteoritos con multiplicadores aleatorios x2, x3, x5 y x10
- Pausa, Game Over y reinicio

Para publicar: sube `index.html` a tu repositorio y activa GitHub Pages.
