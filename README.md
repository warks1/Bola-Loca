# Bola Loca - Playable Web Alpha 0.7

Versión web jugable sin instalación. No necesita Flutter ni servidor.

## Cómo probarlo

1. Descomprime el ZIP.
2. Abre `index.html` en Chrome, Safari, Edge o Firefox.
3. En móvil: arrastra la bola con el dedo.
4. En ordenador: usa ratón o teclado WASD/flechas.

## Incluye

- Menú principal.
- Juego en canvas HTML5.
- Bola controlable.
- Obstáculos procedurales.
- Colisiones.
- Puntuación.
- Multiplicador.
- IA de dificultad simple.
- Bonus de moneda y escudo.
- Skins desbloqueables.
- Guardado local con `localStorage`.
- Pantalla de derrota.
- Funciona en navegador.

## Publicarlo online rápido

Puedes subir estos archivos a GitHub Pages, Netlify, Vercel o cualquier hosting estático.
Solo hacen falta estos tres archivos:

- `index.html`
- `styles.css`
- `game.js`
